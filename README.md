# react-native-midi-playback

MIDI playback in React Native

## Installation

```sh
npm install react-native-midi-playback
```

## Usage

```js
import MidiPlayback from "react-native-midi-playback";

// ...

const result = await MidiPlayback.multiply(3, 7);
```
